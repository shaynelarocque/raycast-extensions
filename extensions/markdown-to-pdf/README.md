# Markdown to PDF

Convert markdown into beautifully styled PDFs using Ask AI. Mention **@Markdown to PDF** in Ask AI, AI Chat, or AI Commands with your markdown content and get a professionally formatted PDF in seconds.

## Features

- **Styled PDF output** — headings, lists, tables, code blocks, blockquotes, and hyperlinks
- **5 built-in themes** — default, minimal, executive, ocean, warm
- **Custom themes** — configure your own accent, text, and surface colors in preferences
- **Layout controls** — font, font size, line spacing, and per-side margins
- **Pure PDF** — no headless browser required, uses pdf-lib for fast, lightweight rendering

## Usage

This is an AI-tool extension. Use it by mentioning **@Markdown to PDF** in:

- **Ask AI** — paste or describe your markdown content
- **AI Chat** — reference it in conversation
- **AI Commands** — use it in your workflows

The AI will generate a PDF and save it to `~/Downloads` (or a directory you specify).

## Options

| Option | Values | Default |
|--------|--------|---------|
| Font | `helvetica`, `times`, `courier` | `helvetica` |
| Font size | `8`–`18` pt | `11` |
| Line spacing | `1.0`–`2.4` | `1.27` |
| Margins | `24`–`120` pt per side | `56` |
| Theme | `default`, `minimal`, `executive`, `ocean`, `warm` | `default` |

## Custom Themes

Configure a custom theme in extension preferences with:

- **Theme name** — use this name when asking the AI to apply your theme
- **Accent color** — hex color for headings, links, and table headers
- **Text color** — hex color for body text
- **Surface color** — hex color for code block and table stripe backgrounds
